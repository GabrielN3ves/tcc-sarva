# SARVA - Serviço de Apoio a Revendedores Autônomos

O **SARVA** é um sistema web integrado desenvolvido como Trabalho de Conclusão de Curso (TCC) para o curso Técnico em Informática do **Colégio Pedro II - Campus Engenho Novo II**. O projeto visa solucionar as dificuldades de revendedores autônomos que realizam seu controle financeiro de forma manual.

##  Sobre o Projeto

Muitos revendedores autônomos gerenciam suas vendas e clientes em cadernos, o que dificulta o controle de inadimplência e a visualização do fluxo financeiro. O SARVA centraliza esses dados, permitindo que o vendedor conheça sua margem de lucro e o histórico de seus clientes para tomar decisões mais seguras.

##  Funcionalidades Principais

* **Gestão de Clientes:** Cadastro e acompanhamento com indicadores visuais de situação financeira (adimplente, inadimplente ou neutro) via bandeiras coloridas.
* **Controle de Vendas e Pedidos:** Registro detalhado de transações com clientes e pedidos feitos às empresas fornecedoras.
* **Gestão de Descontos:** Ferramenta para simular descontos baseados na comissão do revendedor, evitando prejuízos.
* **Relatórios Mensais:** Geração automática de relatórios de entrada, saída, lucros e produtos mais vendidos.
* **Mecanismos de Notificação:** Alertas sobre prazos de faturas e vendas a prazo.
* **Catálogo de Produtos:** Organização de itens por ciclos e empresas.

##  Tecnologias Utilizadas

* **Linguagem:** C#.
* **Web:** HTML, CSS e JavaScript.
* **Banco de Dados:** Microsoft SQL Server.
* **Ambiente de Desenvolvimento:** Visual Studio 2022.
* **Modelagem:** UMLet para diagramas de classe e casos de uso.
